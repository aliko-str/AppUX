Greetings, Evaluator!

This is the main tab; it is pinned and is going to be always open. Don't close before you finished UX evaluation activities.


In this session, you'll evaluate websites and webpages in a few simple steps:

  1. Fill out a short demographics questionnaire.
  2. Read a scenario. Try imagining yourself in the situation that the scenario describes.
  3. A Google-like interface will appear with a list of website links and text snippents. Feel free to explore the websites.

    (All links will open in new tabs.)

  4. Re-arrange the list of websites according to what's said in your scenario. All text snippents can be drag&dropped:

  5. Press "Next" when you're done re-arranging:
  6. Answer a few questions about how you feel.
  7. Answer a few questions about the websites you've explored.
  8. Rate a few more webpages.
  9. The End.
